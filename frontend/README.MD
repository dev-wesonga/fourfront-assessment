#  Membership Page

## Overview

This project recreates the provided design using HTML, CSS, JavaScript, and Bootstrap.

## Features

- Responsive layout (desktop, tablet, mobile)
- Built using Bootstrap grid and components
- Membership description toggle functionality
- Modal interaction
- Basic JavaScript form validation

## Tech Stack

- HTML
- CSS
- JavaScript
- Bootstrap 5

## How to Run

Open `index.html` in your browser.
