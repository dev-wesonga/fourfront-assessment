# Money Tracker API

A Laravel REST API for managing multiple wallets and tracking financial transactions. The system allows users to create accounts, manage different wallets, and record income and expense transactions with real-time balance updates.

## Project Overview

This is a backend API service designed to handle complete wallet management and financial tracking. Users can create accounts, establish multiple wallets for different purposes, and maintain detailed transaction records. The system automatically calculates wallet balances and provides a comprehensive transaction history for each wallet.

## Features

User account creation and management with secure password hashing and email validation. Multiple wallets per user for organizing finances. Income and expense transaction recording with automatic balance calculations. Complete transaction history and filtering capabilities. Real-time balance updates across all wallets. Comprehensive validation on all inputs. Proper error handling with meaningful error messages. Clean and consistent API responses.

## Technology Stack

Framework: Laravel 11
Language: PHP 8.2+
Database: MySQL 8.0+
Authentication: Laravel Sanctum
API Design: RESTful
